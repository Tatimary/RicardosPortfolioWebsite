Why use blockchain?

Blockchain technology offers enhanced security. It does this by using a digital decentralized ledgar. In most cases this ledgar is transparent and verifiable, trustless and secure

Want a career in blockchain? Here's how you can fast track this process!

The following is a list of some important libraries and programming languages to learn in order of importance.

  1. Solidity
  2. Javascript
  3. Web3.js and/or Ethers.js ~ These two libraries allow you to build websites that can communicate with smart contracts
  4. React.js
  5. Node.js or NMP
  6. Express.js
  7. React Native/Expo

Here are also some important frameworks and technologies to get familiar with!

  • Truffle ~An Ethereum smart contract framework!
  • Metamask ~A browser wallet that supports Ethereum tokens and sidechains
  • Ganache ~A blockchain that runs natively on your computer so that you can test your smart contracts before deploying them on a live network.

This may be a very cool tech stack to get familiar with, however what do blockchain devs do with this tech?

blockchain developer is kind of a blanket term, for a person with a skill set that can help build or build on block chain technology such as Ethereum.
Heres a small list of some the roles and responibilities of a blockchain developer.

  • Write Smart Contracts
  • Build Blockchain Websites and UI's
  • Build Mobile Blockchain Applications
  • Build Blockchain Backends and Web Servers ~ An example of this would be displaying real time crypto currency prices on a webpage, or data from a blockchain.
  • Build infrustructure ~ Building blockchains from the ground up, fine tuning or updating existing blockchains.

To get started as a developer there are a few things that you will need to do.

Firstly, you will need to get a development environment setup. To start simply, you can just use a browser based IDE called Remix. However, ultimately if you want to become a professional developer you will need to get a local environment setup. To do this, you will need to get node.js installed through the command line terminal as well as install Truffle.

Some Terminology to Know

Arbitrage is buying crypto on one exchange and selling it on another, with the intent of making money off of the price discrepancies. This can be done using trading bots.
